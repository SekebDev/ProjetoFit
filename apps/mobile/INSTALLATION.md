# 📱 Instalação do Hipertrof.AI

## Android (APK)

### Como Baixar

1. Clique no botão **"Baixar App"** no site
2. O arquivo `hipertrof.apk` será baixado automaticamente

### Como Instalar

#### Método 1: Instalação Direta (Recomendado)

1. Localize o arquivo `hipertrof.apk` no seu gerenciador de arquivos
2. Toque no arquivo para instalar
3. Se receber aviso de "Fontes desconhecidas", confirme:
   - Vá para **Configurações > Segurança > Fontes desconhecidas**
   - Ative "Permitir instalação de apps desconhecidos" (ou similar)
   - Volte e toque no APK novamente
4. Confirme a instalação
5. O app aparecerá em seu launcher com o ícone de Hipertrof.AI

#### Método 2: Via QR Code

Acesse https://hipertrof.ai/install no seu Android para obter um QR code que aponta para o download.

### Primeira Vez

1. Abra o app
2. Faça login com sua conta
3. Permita as permissões solicitadas (câmera é opcional)
4. Comece seus treinos!

## Requisitos

- **Android 8.0+**
- ~50MB de espaço livre
- Conexão com internet (para iniciar; funciona offline depois)

## Problemas?

### "App não instalado"
- Verifique se há espaço suficiente no telefone
- Tente deletar arquivos temporários e reinstalar
- Confirmeseu Android é 8.0+

### "Fontes desconhecidas bloqueadas"
- Siga as instruções acima em "Método 1"
- Diferentes fabricantes têm nomes diferentes; procure por "Apps de origem desconhecida"

### Não consigo fazer login
- Verifique sua conexão com internet
- Confirme email e senha estão corretos
- Tente novamente em alguns minutos

## Atualizações

Novas versões do app serão disponibilizadas no botão "Baixar App" do site.
Quando houver atualização:
1. Baixe o novo APK
2. Toque no arquivo para atualizar
3. Seu progresso será preservado
